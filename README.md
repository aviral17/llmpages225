# Web Resource Archive

This is a simple, responsive web application that provides access to a variety of files including a short story, JSON data files, an SVG image, and a description about the creator.

## Features
- Download links for each resource
- Clean, mobile-friendly design built with Tailwind CSS
- Easy to extend with additional resources

## Deployment
You can host this in GitHub Pages by pushing this repository and enabling GitHub Pages in the repository settings.

## Files Included
- `index.html`: Main web page displaying resource links.
- `ashravan.txt`: Short story about Ashravan.
- `dilemma.json`: Ethical dilemma data for autonomous vehicles.
- `about.md`: Brief description of the author.
- `pelican.svg`: SVG illustration.
- `restaurant.json`: Restaurant recommendation in Chennai.
- `prediction.json`: Economic forecast JSON.
- `uid.txt`: Your UID file.
- `LICENSE`: MIT License text.
